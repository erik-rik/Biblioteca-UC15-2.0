# Sistema de Biblioteca

## Status do Projeto
- Em desenvolvimento

## Tecnologias Utilizadas
- Java  
- Swing (para interface gráfica)  
- JDBC (para conexão com o banco de dados)  
- MySQL (banco de dados relacional)

## Time de Desenvolvedores
- Erik Richter Knabach

## Objetivo do Software
- O sistema tem como objetivo auxiliar no gerenciamento de uma biblioteca, permitindo o cadastro, consulta e controle de livros, usuários e empréstimos, de forma prática e organizada.

## Funcionalidades do Sistema
- Cadastro de livros (título, autor, categoria etc.)  
- Cadastro de usuários (nome, matrícula, contato etc.)  
- Registro de empréstimos de livros para usuários  
- Controle de devoluções (data prevista e data de entrega)  
- Consulta de livros disponíveis e emprestados  
- Atualização e exclusão de registros (livros, usuários e empréstimos)  
- Interface gráfica simples e intuitiva
